# Data Materials

This contains:

* `TaxFeatures_Template.xml`, the XML schema used to organise the tax features examined in the study; and
* `data_dictionary.md`, which defines the XML tags used in that schema.

The repository does not include the underlying tax-feature values, completed country profiles, source texts, or other proprietary materials. The underlying values were derived from the IBFD and PwC databases and remain subject to the respective providers’ terms of use.

Researchers with lawful access to those sources may use the schema and data dictionary to populate the tagged fields and to reproduce, verify, extend, or adapt the dataset.
